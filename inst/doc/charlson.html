Introduction

The medicalcoder package implements several variants of the Charlson comorbidities algorithm.

  • charlson_deyo1992: Deyo’s original set of codes (Deyo et al. 1992; Quan et al. 2005)
  • charlson_quan2005 and charlson_quan2011: Codes and index scoring (Quan et al. 2005, 2011)
  • charlson_cdmf2019: (Glasheen et al. 2019)

Applying Charlson

Example: applying the Quan et al. (2005) variant of the Charlson comorbidities to the mdcr data can be done as follows.

mdcr_results <-
  comorbidities(
    data = mdcr,
    id.vars = "patid",
    icdv.var = "icdv",
    icd.codes = "code",
    dx.var = "dx",
    flag.method = "current",
    poa = 1,
    primarydx = 0,
    method = "charlson_quan2005"
  )

The return object is a data.frame with 0/1 integer indicator columns for the relevant conditions, the id.vars (if applicable), age_score (if age, in years, is supplied), num_cmrb the number of comorbidities, cmrb_flag a 0/1 indicator for presence of at least one comorbidity, and cci the Charlson Comorbidity Index.
